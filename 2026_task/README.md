**命名规则**
- 统一放在以adapt开头的文件夹中
- 文件夹后字母L/R/M分别代表蓝色和红色的位置
    - L:左侧
    - R:右侧
    - M:中间

- 由规则，可能出现的情况有：
    - LL
    - LR
    - RL
    - RR
    - MM
